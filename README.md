功能说明：
多格式支持：您可以直接上传图片或视频。

图片智能擦除：

在加载图片后，使用鼠标（或触摸）在图片上框选水印区域。

点击“立即擦除”，程序会使用像素采样填充算法覆盖该区域，并自动进行边缘平滑处理。

视频水印遮盖：

由于浏览器端无法直接逐帧重绘视频编码，程序提供了一个可拖动的半透明黑色遮罩。

您可以将遮罩移动到视频中 Gemini 或其他 AI 的水印位置。

离线安全：所有操作均在您的浏览器本地完成，不会将您的私人媒体文件上传到服务器。

一键下载：处理完成后，可以直接导出处理好的图片（视频建议使用遮罩位置记录配合专业剪辑软件裁剪）。

使用技巧：
针对 Gemini 水印：Gemini 生成的水印通常在右下角。框选时，尽量只框选文字部分，这样周围像素填充的效果会更自然。

视频处理建议：如果视频水印在黑边区域，可以简单地通过本工具的遮罩预览效果，确认遮挡位置后再进行保存。
